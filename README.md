# Msc_Network_Analysis_of_the_Danish_and_Italian_Music_Industries
A database created for the paper "Network Analysis of the Danish and Italian Music Industries" by Daniel Fejerskov-Quist and Lisandro M. Benetti (2025).
It includes a bipartite network and projeced network of the Danish music industry and an bipartite network and projected network of the Danish network merged with the Italian network presented in Node Attribute Analysis for Cultural Data Analytics: a Case Study on Italian XX-XXI Century Music by Michele Coscia (2024). His database can be downloaded from https://www.michelecoscia.com/
